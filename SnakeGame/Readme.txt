#Snake Game
Created using html, css and javaScript

#Rules
The Snake eats food and grows in size.
You can change the direction of the Snake to redirect it towards the food.
The score increases by +1 each time the snake eats a food item
If the snake hits its body, you lose.

#How to start

Dowload or clone the files into your system
Then start playing by opening the index.html file
Use 'Up', 'Down', 'Right', 'Left' keys to change the direction of the Snake
